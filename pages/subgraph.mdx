# Subgraph

Subgraphs can be found on [Github](https://github.com/0xSplits/splits-subgraph)
and can be explored via GraphQL playgrounds for the following networks.

<Callout>
  All subgraph queries are **case sensitive** and must be **lowercase**.
</Callout>
