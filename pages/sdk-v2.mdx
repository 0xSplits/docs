# Splits Development Kit

This page explains how to use the old ethers version of the **Splits Development
Kit (SDK)**. The SDK makes it easier to query the [subgraph](/subgraph) or call
[contract functions](/core). You can find the SDK source code on
[Github](https://github.com/0xSplits/splits-sdk/tree/v2).

To get started, install the package using `yarn` or `npm`.

<Tabs items={['yarn', 'npm']}>

  <Tab>

```bash
yarn add @0xsplits/splits-sdk@2
```

  </Tab>
  <Tab>

```bash
npm install @0xsplits/splits-sdk@2
```

  </Tab>
</Tabs>
