# Splits Development Kit

This page explains how to use the **Splits Development Kit (SDK)**. The SDK
makes it easier to query the [subgraph](/subgraph) or call
[contract functions](/core). You can find the SDK source code on
[Github](https://github.com/0xSplits/splits-sdk).

To get started, install the package using `yarn` or `npm`.

<Tabs items={['yarn', 'npm']}>

  <Tab>

```bash
yarn add @0xsplits/splits-sdk
```

  </Tab>
  <Tab>

```bash
npm install @0xsplits/splits-sdk
```

  </Tab>
</Tabs>

The SDK uses viem under the hood. If you are using ethers-v5 and do not want to add viem to your project,
you can check out the [original version of the SDK](/sdk-v2) which uses ethers-v5.

If you are migrating from the ethers version to the viem version, please see the
[migration notes](https://github.com/0xSplits/splits-sdk/releases/tag/%400xsplits%2Fsplits-sdk%403.0.0).
